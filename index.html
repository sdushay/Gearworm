GEARWORM -- Developed by Social Experiment Games 2.0 (Dushay, Russell, Jono, Fred)

Enjoy the game! Works on Android and PC, maybe iOS coming soon. Swipe to move, try to get three of the same color in a row for some extra points and a shorter tail. It was made using Kivy, a multiplatform game engine in python. For the Kivy homepage, go Here

This game was made as a final project for Ming Chow's Game Development Course, Spring 2012 Tufts University
